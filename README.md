# 🏗 МособлГосЭкспертиза — DocumentAnalyzer v2.0 (WEB)

Автоматизированная система приёмки и проверки проектной документации (ПД) для государственной строительной экспертизы.

**Web-интерфейс** позволяет загружать ZIP-архивы или отдельные файлы через браузер, проводит 8-этапный анализ (ПП РФ №963, №154, НОПРИЗ, RAG) и генерирует отчеты в реальном времени.

---

## 🚀 Быстрый старт

```bash
# 1. Клонировать
git clone https://github.com/14Segun88/MOGE_TG_24_03_26.git
cd MOGE_TG_24_03_26

# 2. Установить (venv, зависимости, Weaviate)
chmod +x install.sh && ./install.sh

# 3. Настроить ключи
cp .env.example .env
nano .env   # Обязательно: GROQ_API_KEY, LM_STUDIO_API_KEY, LM_STUDIO_URL

# 4. Запустить Web-интерфейс
.venv/bin/python web_app.py
```
Зайдите по адресу: [http://localhost:8000](http://localhost:8000)

### Требования
- **ОС:** Linux (Ubuntu 22.04+) / WSL2
- **Python:** 3.10
- **Docker:** для работы Weaviate (векторная БД)
- **LM Studio** (на Windows): модель `nomic-embed-text` для RAG-эмбеддингов
- **Groq API Key:** для работы Llama-3.3-70B (llama-3.3-70b-versatile)

## 📁 Ключевые компоненты v2.0

### Интерфейс и Сервер
*   `web_app.py`: Основной сервер на FastAPI. Поддерживает WebSockets для работы в реальном времени.
*   `web/index.html`: Чат-интерфейс с поддержкой Drag-n-drop и вложений.
*   `start.sh`: Скрипт проверки состояния и запуска системы.

### Аналитика и Проверки
*   `test_first_page.py`: Продвинутая сверка титульных листов. Поддерживает:
    *   **PDF**: Текстовые и сканы (через OCR `pytesseract`).
    *   **XML**: Автоматический разбор метаданных по XSD.
    *   **IFC (BIM)**: Извлечение данных о проекте, этажности и авторе.
*   `rag_indexer.py`: Скрипт индексации нормативной базы в Weaviate (55 документов).
*   `project_search.py`: Агент поиска по базе знаний и нормативным документам.
*   `src/api/pipeline.py`: Сердце системы — управление 8 специализированными агентами.

## 🛠 Новые возможности (Март 2026)
1. **BIM Integration (IFC)**: Теперь система понимает BIM-модели и сверяет их метаданные с остальным пакетом.
2. **OCR для сканов**: При обнаружении PDF без текстового слоя подключается OCR (DPI >= 300).
3. **Автоматическая сверка XML**: Распознавание версий 01.05 и 01.06 (Приказ 421/пр).
4. **Улучшенные синонимы**: Разделы ИОС1-ИОС7 теперь определяются строго по ПП РФ №87.
